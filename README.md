aladdins_escape
==============

The game is developed as a final project for Intro to CS. The game follows Alladin in his quest to escape a collapsing cave. 

Features and specs
Resolution: 533x300
FPS: 30
Number of levels: 5
Menu: yes
Music: yes
SFX: yes

PyGame module required!
